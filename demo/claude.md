# react-key-context Demo

This is a demonstration application for the `react-key-context` library.

## Running the Demo

1. Install dependencies: `pnpm install`
2. Start development server: `pnpm dev`
3. Build for production: `pnpm build`

## Purpose

Showcases the features of `react-key-context`:

- Key-based context management.
- Nested providers and shadowing.
- Type-safe hooks.
